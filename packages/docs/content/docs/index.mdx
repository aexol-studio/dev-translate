# Welcome to dev-translate

**dev-translate** is an AI-powered internationalization (i18n) tool that automatically translates your JSON locale files. It works with your existing i18n setup and integrates seamlessly with popular frameworks.

## Why dev-translate?

Translating your application to multiple languages is time-consuming and expensive. dev-translate uses AI to:

- **Automatically translate** your JSON locale files to 30+ languages
- **Preserve context** with smart translation that understands your application
- **Save costs** with incremental translation - only translate what changed
- **Integrate seamlessly** with Vite, Next.js, or any build tool via CLI

## How It Works

1. **Configure** - Set up your source language and API key
2. **Organize** - Put your source translations in a locale folder (e.g., `/locales/en/`)
3. **Translate** - Run the CLI or use a framework plugin
4. **Deploy** - Your translations are ready in all target language folders

```
locales/
├── en/           # Source language (you write this)
│   ├── common.json
│   └── auth.json
├── de/           # Auto-generated
│   ├── common.json
│   └── auth.json
├── fr/           # Auto-generated
│   ├── common.json
│   └── auth.json
└── es/           # Auto-generated
    ├── common.json
    └── auth.json
```
